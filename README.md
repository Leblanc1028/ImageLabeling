# ImageLabeling
自製圖片標註程式
以C#(WPF)語言製作的影像標註程式，同時練習前端與後端開發。
為了使程式碼利於優化及維護，95%版面元件及元件中的事件觸發皆為自製，可以隨開發者需求增減想要的事件。
